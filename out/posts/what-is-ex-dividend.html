# 什麼是除權息？投資人必懂的股票分派概念

## 前言

每年到了配息季節，股票會經歷「除權息」。這是什麼意思？對投資人有什麼影響？

## 除權息定義

除權息是指公司將盈餘分配給股東的行為，分為「除權」和「除息」兩種：

- **除息**：發放現金股利
- **除權**：發放股票股利

## 除權 vs 除息

| 比較項目 | 除息 | 除權 |
|----------|------|------|
| 發放形式 | 現金 | 股票 |
| 股價變化 | 扣除現金金額 | 扣除股票價值 |
| 股東權益 | 減少（現金流出） | 不變（股本增加） |
| 持股數量 | 不變 | 增加 |

## 對股價的影響

### 除息參考價
```
除息參考價 = 前一日收盤價 - 每股配息金額
```

### 除權參考價
```
除權參考價 = 前一日收盤價 ÷ (1 + 配股率)
```

### 範例
某股票收盤價100元：
- 除息2元：參考價 = 100 - 2 = 98元
- 除權1元：參考價 = 100 ÷ (1 + 0.1) = 90.9元

## 投資人該注意什麼

### 重要日期
1. **宣佈日**：公司宣佈配息配股
2. **除權息日**：股價調整的日期
3. **基準日**：決定誰能領股利的日期
4. **發放日**：實際發放股利的日期

### 填權填息
- **填息**：股價回升到除息前價格
- **填權**：股價回升到除權前價格
- **貼息/貼權**：股價未回升

### 節稅考量
- 可選擇分開計稅（28%）或併入綜所稅
- 高所得者可考慮分開計稅

## 結論

除權息是公司回饋股東的方式。了解除權息機制，能幫助投資人做出更好的決策。
