# 股票可以當退休金嗎？用股票打造退休收入的完整規劃

## 前言

隨著勞保、勞退面臨改革壓力，越來越多人開始思考：股票可以當退休金嗎？

## 股票當退休金的可行性

### 歷史報酬率
- S&P 500 長期年化報酬率約 7-10%
- 台股 0050 長期年化報酬率約 8-10%

### 複利效應
時間是投資最好的朋友。長期持有，複利效果會讓資產大幅成長。

### 稅務優惠
- 股利所得可選擇分開計稅或併入綜所稅
- 長期持有享受稅務優惠

## 優缺點分析

### 優點
1. 資產增值潛力高
2. 股息收入穩定
3. 流動性佳
4. 對抗通膨

### 缺點
1. 市場波動風險
2. 需要時間累積
3. 選股能力要求
4. 情緒管理挑戰

## 如何規劃（五步驟）

### 步驟一：設定目標
確定退休年齡、生活水準、年度支出需求。

### 步驟二：計算規模
用 4% 法則計算退休所需資產：
```
退休資產 = 年度支出 ÷ 4%
```

### 步驟三：選擇策略
- ETF 策略：0050 + 0056 + 00878
- 高股息個股：中華電信、玉山金等
- 全球分散：VOO + VT

### 步驟四：定期投入
每月定期定額，長期持有。

### 步驟五：建立提領機制
退休後每年提領 4%，保持資產永續。

## 注意風險

1. **時間風險**：退休初期市場大跌
2. **集中風險**：過度集中單一標的
3. **通膨風險**：購買力下降
4. **長壽風險**：活太久錢不夠用
5. **市場時序風險**：進場時點不佳

## 不同年齡層建議

| 年齡 | 建議配置 |
|------|----------|
| 30-40歲 | 70%股票 + 30%債券 |
| 40-50歲 | 60%股票 + 40%債券 |
| 50-60歲 | 50%股票 + 50%債券 |
| 60歲以上 | 40%股票 + 60%債券 |

## 結論

股票可以作為退休收入的重要來源，但需要長期規劃與風險管理。越早開始，複利效果越顯著。
