# Blog UI 設計規格 — 參考 FreeDidi 風格

## 一、整體風格分析（參考 FreeDidi.com）

### 1. 頁面結構
- 頂部導航列（Logo + 分類選單）
- 大尺寸 Hero Banner（首頁特色文章）
- 卡片式文章列表（Grid Layout）
- 側邊欄（熱門文章 + 廣告位）
- 底部資訊區（關於我們、聯絡方式）

### 2. 廣告位置配置
| 位置 | 格式 | 尺寸 |
|------|------|------|
| 頂部導航下方 | 橫幅廣告 | 728x90 |
| 文章列表間隙 | 插頁廣告 | 300x250 |
| 側邊欄 | 直式廣告 | 300x600 |
| 文章內容中 | 原生廣告 | 響應式 |
| 文章底部 | 橫幅廣告 | 728x90 |

## 四、響應式設計斷點

| 裝置 | 寬度 | 版面配置 |
|------|------|----------|
| 桌面 | ≥1024px | 3欄（主內容 + 側邊欄） |
| 平板 | 768-1023px | 2欄（主內容 + 簡化側邊欄） |
| 手機 | <768px | 單欄（側邊欄移至底部） |

## 五、配色方案

### 主要顏色（財經專業感）
- 主色：#1a56db（深藍色）
- 次色：#10b981（綠色 — 代表成長）
- 背景：#ffffff（白色）
- 文字：#1f2937（深灰色）
- 次要文字：#6b7280（灰色）

### 廣告區域邊框
- 廣告容器：#f3f4f6（淺灰背景）
- 廣告標籤：「廣告」文字

---

## 六、廣告配置策略

### 廣告密度控制
- 首頁：每 6 篇文章插入 1 個廣告卡片
- 文章頁：每 4-5 個段落插入 1 個原生廣告
- 側邊欄：固定 1 個直式廣告

### 廣告類型
1. **Google AdSense**：自動廣告
2. **原生廣告**：融入內容的推薦文章
3. **聯盟行銷**：投資工具推薦（券商、投資App）
