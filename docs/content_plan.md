# SEO 內容計畫 — Blog 財經/投資網站

## 1. 內容策略總覽

### 1.1 目標
- 建立財經/投資領域的權威內容
- 透過 SEO 獲取自然搜尋流量
- 達成日均 US$5 收入目標

### 1.2 內容策略
- **主題集群（Topic Cluster）**：建立核心主題與衛星文章
- **長尾關鍵字**：鎖定低競爭、高轉換的關鍵字
- **內容改寫**：搜尋整合現有內容，改寫為原創文章
