# 什麼是信用交易？融資融券完整解析

## 前言

開戶時常看到「信用交易」選項，這是什麼？風險有多高？

## 信用交易定義

信用交易是指向券商借錢買股（融資）或借股票賣出（融券），具有槓桿效應。

### 槓桿效應
- 放大獲利：上漲時賺更多
- 放大虧損：下跌時虧更多

## 融資是什麼

### 定義
向券商借錢買股票，放大投資槓桿。

### 運作方式
- 自備款：40%
- 融資：60%
- 利息：年約6%

### 範例
買進10萬元股票：
- 自備款：4萬元
- 融資：6萬元
- 槓桿倍數：2.5倍

### 計算
```
股票上漲10% → 報酬率 = 25%（放大）
股票下跌10% → 虧損率 = 25%（放大）
```

## 融券是什麼

### 定義
向券商借股票賣出，看跌時使用。

### 運作方式
- 借股票賣出
- 等股價下跌後買回
- 歸還股票並賺取差價

### 注意事項
- 有強制回補風險（除權息前）
- 上漲時虧損無限

## 風險分析

### 一、槓桿風險
虧損放大，可能賠光本金。

### 二、追繳斷頭風險
維持率低於一定比例會被追繳或強制平倉。

### 三、利息成本
融資需支付利息（年約6%）。

### 四、流動性風險
市場劇烈波動時可能無法即時賣出。

| 風險 | 融資 | 融券 |
|------|------|------|
| 槓桿風險 | 虧損放大 | 無限虧損 |
| 追繳風險 | 有 | 有 |
| 利息成本 | 有 | 有 |
| 強制平倉 | 有 | 有 |

## 結論

信用交易風險高，新手不建議使用。充分了解風險後再考慮，並嚴格控制槓桿倍數。
