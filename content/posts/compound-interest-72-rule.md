# 複利效果如何計算？用72法則快速估算

## 前言

想知道投資多久能翻倍？72法則是最簡單的工具。

## 複利定義

複利是指「利滾利」，投資收益再投入，讓本金和收益一起成長。

## 72法則說明

72法則公式：
```
資產翻倍時間 ≈ 72 ÷ 年化報酬率
```

## 計算範例

### 範例一：年化報酬7%
翻倍時間 = 72 ÷ 7 = **約10年**

### 範例二：年化報酬10%
翻倍時間 = 72 ÷ 10 = **約7年**

### 範例三：年化報酬5%
翻倍時間 = 72 ÷ 5 = **約14年**

## 實際應用

| 年化報酬率 | 翻倍時間 |
|------------|----------|
| 3% | 24年 |
| 5% | 14年 |
| 7% | 10年 |
| 10% | 7年 |
| 15% | 5年 |

## 結論

72法則讓你快速估算複利效果。記住：年化7%的投資，10年就能翻倍！
