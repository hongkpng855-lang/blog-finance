# 什麼是除權息？投資人必懂的股票分派概念

## 前言

每年配息季節，股票會經歷「除權息」。這是什麼意思？對投資人有什麼影響？

## 除權息的定義

除權息是指公司將盈餘分配給股東的行為，分為「除權」和「除息」兩種。

## 除權 vs 除息

### 除息（現金股利）
公司發放現金給股東。

**除息參考價公式：**
```
除息參考價 = 前一日收盤價 - 每股配息金額
```

### 除權（股票股利）
公司發放股票給股東，增加股東持股數。

**除權參考價公式：**
```
除權參考價 = 前一日收盤價 ÷ (1 + 配股率)
```

### 綜合效果（除權息）
同時發放現金股利與股票股利。

## 對股價影響

### 股價調整機制
除權息當天，股價會自動調整，反映已發放的股利。

### 填權息
- **填息**：股價回升到除息前價格
- **填權**：股價回升到除權前價格
- **貼息/貼權**：股價未回升

### 市場心理
公司能否填權息，反映市場對其未來的看法。

## 投資人該注意什麼

### 基本面
公司是否具備長期競爭力。

### 股利政策
配息是否穩定、配發率是否合理。

### 稅負考量
- 可選擇分開計稅（28%）或併入綜所稅
- 高所得者可考慮分開計稅

### 殖利率
配息金額 ÷ 股價 = 現金殖利率

### 重要時程
1. 宣佈日
2. 除權息日
3. 基準日
4. 發放日

## 結論

除權息是公司回饋股東的方式。了解除權息機制，能幫助投資人做出更好的決策。
