# 什麼是股價淨值比？如何用P/B判斷股價是否合理

## 前言

在投資評估工具中，股價淨值比（P/B）是判斷股價是否合理的重要指標之一。

## 定義

股價淨值比（Price-to-Book Ratio，簡稱 P/B）是股價與每股淨值的比值，反映投資人願意為公司淨資產支付多少倍價格。

## 計算公式

```
P/B = 股價 ÷ 每股淨值
```

### 實例說明
某公司股價 50 元，每股淨值 40 元：
```
P/B = 50 ÷ 40 = 1.25 倍
```

## 如何判斷

### 歷史比較
與公司自身的歷史 P/B 比較，判斷目前處於相對高點或低點。

### 同業比較
與同產業公司的 P/B 比較，評估相對便宜或昂貴。

### P/B < 1 的意義
股價低於淨值，可能被低估，但也需注意公司是否有問題。

### 搭配其他指標
建議搭配本益比、殖利率、ROE 等指標綜合判斷。

## 與本益比差異

| 比較項目 | 本益比（P/E） | 股價淨值比（P/B） |
|----------|---------------|-------------------|
| 評估基礎 | 盈餘 | 淨值 |
| 適用產業 | 獲利公司 | 所有公司 |
| 無形資產敏感度 | 低 | 高 |
| 適合投資策略 | 成長股 | 價值股 |

## 結論

股價淨值比是評估股價的重要工具，適合用於價值型投資判斷。建議搭配其他指標綜合使用。
