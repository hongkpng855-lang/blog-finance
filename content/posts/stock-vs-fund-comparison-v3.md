# 股票和基金有什麼不同？完整比較分析

## 前言

投資股票和買基金有什麼不同？哪個適合你？

## 股票的定義

股票是公司所有權憑證。買進股票就成為股東，享有公司獲利的分配權。

### 股票特性
- 需自行研究分析
- 直接持有公司股份
- 可參與股東會投票

## 基金的定義

基金是由投信公司募集資金，由專業經理人投資一籃子標的。

### 基金特性
- 專業經理人管理
- 匯集眾人資金
- 投資組合分散

## 兩者差異比較

| 比較項目 | 股票 | 基金 |
|----------|------|------|
| 投資門檻 | 一張起跳 | 小額即可 |
| 專業程度 | 需要研究能力 | 不需專業知識 |
| 風險程度 | 集中風險高 | 分散風險低 |
| 投資彈性 | 高度自主 | 受基金契約限制 |
| 管理費用 | 交易手續費 | 經理費+手續費 |
| 收益來源 | 股利+價差 | 淨值增長+配息 |
| 投資標的 | 單一公司 | 一籃子標的 |
| 時間投入 | 需要時間研究 | 省時方便 |

## 適合投資股票的對象

1. 願意花時間研究的人
2. 能承受較高風險
3. 追求較高報酬
4. 有投資經驗
5. 喜歡自主決策

## 適合投資基金的對象

1. 投資新手
2. 沒時間研究
3. 希望分散風險
4. 追求穩健報酬
5. 喜歡省時方便

## 結論

股票和基金各有優勢。建議採用「核心-衛星策略」：
- 核心（70-80%）：基金或ETF
- 衛星（20-30%）：個股
