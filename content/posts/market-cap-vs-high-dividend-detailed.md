# 市值型ETF vs 高股息ETF：差異比較與選擇建議

## 前言

台股ETF主要分為兩大類：市值型ETF和高股息ETF。兩者各有特色，適合不同投資人。

## 市值型ETF定義

市值型ETF以「公司市值」為選股依據，追蹤大盤指數。

### 代表ETF
- 0050（元大台灣50）
- 006208（富邦台50）

### 特點
| 項目 | 說明 |
|------|------|
| 選股依據 | 市值大小 |
| 著重目標 | 資本增值 |
| 成分股 | 大型權值股 |
| 波動性 | 相對較小 |
| 殖利率 | 2%-3% |

## 高股息ETF定義

高股息ETF以「現金股利殖利率」為選股依據，著重配息收入。

### 代表ETF
- 0056（富邦高股息）
- 00878（國泰ESG永續高股息）
- 00919（群益台灣精選高息）

### 特點
| 項目 | 說明 |
|------|------|
| 選股依據 | 股利殖利率 |
| 著重目標 | 現金流收入 |
| 配息頻率 | 季配息或年配息 |
| 波動性 | 可能較大 |
| 殖利率 | 4%-7% |

## 兩者差異比較

| 比較項目 | 市值型ETF | 高股息ETF |
|----------|-----------|-----------|
| 選股邏輯 | 市值排序 | 殖利率排序 |
| 長期報酬 | 著重成長 | 著重配息 |
| 適合族群 | 追求資產成長 | 需要現金流 |
| 風險特性 | 波動較小 | 波動可能較大 |

## 適合對象分析

### 適合市值型ETF
- 追求資產成長的投資人
- 不需要穩定現金流
- 長期投資規劃（10年以上）
- 願意承受合理波動

### 適合高股息ETF
- 需要穩定現金流的投資人
- 退休或半退休族群
- 偏好定期領息的感受
- 對波動較敏感

## 配置建議

建議採取「核心—衛星」配置：
- **核心（60%-70%）**：市值型ETF，追求長期成長
- **衛星（30%-40%）**：高股息ETF，提供現金流

## 結論

市值型ETF適合追求成長的投資人，高股息ETF適合需要現金流的投資人。兩者可以搭配配置，兼顧成長與收益。
