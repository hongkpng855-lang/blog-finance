# 資產配置黃金比例：100減年齡法則

## 前言
「資產配置決定90%的投資報酬」——這句投資界的名言點出了資產配置的重要性。而「100減年齡」法則，正是幫助投資人根據年齡調整風險的經典策略。

## 100減年齡法則是什麼？
公式很簡單：
```
股票配置比例 = 100 - 你的年齡
債券配置比例 = 你的年齡
```

### 範例
- 30歲：股票70%、債券30%
- 40歲：股票60%、債券40%
- 50歲：股票50%、債券50%
- 60歲：股票40%、債券60%

## 為什麼要這樣配置？
- 年輕時風險承受度高，可增加股票比重追求成長
- 年長時需要穩定現金流，應增加債券比重降低波動

## 現代調整版本
考慮到現代人壽命延長、利率環境變化，有些專家建議將公式改為：
```
股票配置比例 = 110 或 120 - 你的年齡
```

## 執行步驟
1. 計算自己的股票與債券比例
2. 選擇適合的股票ETF（如0050、VTI）
3. 選擇適合的債券ETF
4. 定期再平衡（每年一次）

## 結論
100減年齡法則提供了一個簡單易懂的資產配置起點。但每個人的財務狀況不同，應根據自身風險承受度、投資目標適度調整。
