# 什麼是股價淨值比？如何用P/B判斷股價是否合理

## 前言

除了本益比，股價淨值比（P/B）也是評估股價的重要指標。

## 股價淨值比定義

股價淨值比（Price-to-Book Ratio，簡稱P/B或PBR）是股價與每股淨值的比值，反映投資人願意為公司的淨資產支付多少倍價格。

## 計算公式

```
股價淨值比 = 股價 ÷ 每股淨值
```

### 範例
某公司股價50元，每股淨值40元：
股價淨值比 = 50 ÷ 40 = **1.25倍**

## 如何判斷

### 一般標準
| P/B值 | 解讀 |
|-------|------|
| < 1倍 | 股價低於淨值（可能被低估） |
| 1-1.5倍 | 合理區間 |
| > 2倍 | 可能偏高 |

### 注意事項
- 不同產業P/B標準不同
- 資產密集產業（銀行、保險）P/B較低
- 輕資產產業（科技、軟體）P/B較高

## 與本益比的差異

| 比較項目 | 本益比（P/E） | 股價淨值比（P/B） |
|----------|---------------|-------------------|
| 比較基礎 | 盈餘 | 淨值 |
| 適用情況 | 獲利公司 | 所有公司 |
| 產業差異 | 成長股較高 | 資產股較低 |
| 意義 | 獲利能力 | 資產價值 |

## 實務應用

### 適合用P/B的情況
- 公司獲利不穩定
- 景氣循環股
- 資產密集產業
- 金融股

### 不適合的情況
- 無形資產占比高的公司
- 高成長科技公司

## 結論

股價淨值比是評估股價的另一個角度。P/B < 1倍時可能被低估，但要考慮產業特性。建議搭配本益比綜合判斷。
