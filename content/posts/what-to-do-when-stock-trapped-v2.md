# 股票被套牢怎麼辦？三種應對策略

## 前言

「股票套牢了怎麼辦？」這是許多投資人都曾面臨的難題。

## 套牢是什麼？

套牢是指買進股票後股價下跌，帳面虧損嚴重，投資人不願意或無法在當前價格賣出的狀態。

### 套牢的類型

| 類型 | 定義 |
|------|------|
| 淺套 | 虧損10-20% |
| 深套 | 虧損30%以上 |
| 良性套牢 | 公司基本面良好 |
| 惡性套牢 | 公司基本面惡化 |

## 三種應對策略

### 策略一：加碼攤平

**做法**：在低點加碼買入，降低平均成本

**優點**：
- 降低持有成本
- 反彈時更快回本

**缺點**：
- 可能愈攤愈平
- 需要額外資金

**適用情況**：
- 公司基本面良好
- 股價下跌是暫時性的
- 有閒置資金可運用

### 策略二：耐心等待

**做法**：不做任何操作，等待股價回升

**優點**：
- 不需要額外資金
- 避免愈攤愈平

**缺點**：
- 時間成本高
- 可能永遠無法回本

**適用情況**：
- 公司體質健全
- 不急需用錢
- 願意長期等待

### 策略三：停損出場

**做法**：認賠賣出，將資金轉往其他機會

**優點**：
- 及時止損
- 釋放資金

**缺點**：
- 實現虧損
- 可能賣在低點

**適用情況**：
- 公司基本面惡化
- 有更好的投資機會
- 需要控制風險

## 如何選擇？

評估四個面向來決定：

1. **公司基本面**：是否仍具投資價值？
2. **資金狀況**：是否有閒置資金？
3. **時間成本**：能等多久？
4. **風險承受度**：能承受多大虧損？

| 情況 | 建議策略 |
|------|----------|
| 基本面良好 + 有資金 | 加碼攤平 |
| 基本面良好 + 無資金 | 耐心等待 |
| 基本面惡化 | 停損出場 |
| 不確定 | 部分停損部分持有 |

## 結論

套牢不可怕，可怕的是沒有策略。根據公司基本面、資金狀況和風險承受度，選擇最適合的應對方式。
